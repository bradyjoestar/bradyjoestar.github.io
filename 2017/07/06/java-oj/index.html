写在前面

lintcode 与 leetcode 刷题的好处：只关心功能的实现，不用处理相应的输入和输出。

对于大多数程序员而言这个功能非常好，处理输入和输出有时候也需要消耗一定的时间，尤其遇到golang这种特例的时候。

刷题需要掌握的基本知识点如下：

  • java 数组、list及初始化
  • java 对象的引用和指针
  • treeNode
  • java int最大值最小值表示法
  • lintcode 图论问题举例
