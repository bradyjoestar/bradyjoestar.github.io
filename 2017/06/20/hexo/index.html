description

本博客是基于hexo主题搭建出来的，每次编写marddown文件后通过hexo deploy同步更新到github上，通过bradyjoestar.github.io 访问。

在搭建过程中主要遇到的问题是git bash的用户切换问题。详细描述如下：

git bash默认用户为swbsin，而生成博客的用户为bradyjoestar，为了解决这种问题：

  • 将swbsin的私钥放入到bradyjoestar的github上面。
  • 每次hexo deploy后生成的public文件复制到到github的bradyjoestar.github.io本地仓库上，并进行上传。

advantage

这样部署博客主要有几个优点：

  • 主题多样，页面简洁。
  • markdown文件对与程序员而言非常优雅，无需使用网站自带的富文本框编辑器。
  • 项目托管在github上，无需另外再租服务器。

tips

hexo的_config.yml配置起来比较复杂，为了简便，可以直接使用Asina的yml文件
